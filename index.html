Política de Privacidade

Data de Vigência: 6 de abril de 2025

A Topcons respeita sua privacidade e está comprometida com a proteção de dados cadastrados no aplicativo Topcons Itacaré Imóveis. Esta política descreve como suas informações são coletadas, utilizadas e armazenadas.

1. Informações Coletadas

  • Dados de cadastro de imóveis (matrícula, proprietário, área, localização)
  • Imagens dos imóveis
  • Arquivos KML
  • Dados de geolocalização

2. Como as informações são usadas

As informações são utilizadas para gerenciamento de imóveis, visualização em mapa e armazenamento seguro no Firebase.

3. Armazenamento de dados

Todos os dados são armazenados de forma segura nos serviços da plataforma Google Firebase (Firestore e Storage).

4. Compartilhamento de dados

Os dados não são compartilhados com terceiros, exceto se exigido por lei.

5. Segurança

Aplicamos medidas técnicas para proteger os dados. No entanto, nenhum sistema é 100% seguro.

6. Direitos dos usuários

Os dados podem ser atualizados, editados ou excluídos diretamente no app pelos operadores autorizados.

7. Alterações nesta política

Esta política pode ser atualizada. A versão mais recente estará sempre disponível neste link.
